
القوانين الأساسية للجبر
قوانين الجمع والطرح
a + b = b + a

خاصية التبديل للجمع: ترتيب الجمع لا يغير من النتيجة

(a + b) + c = a + (b + c)

خاصية التجميع للجمع: يمكن تجميع الحدود بأي ترتيب

a + 0 = a

العنصر المحايد للجمع: إضافة الصفر لا تغير القيمة

a + (-a) = 0

النظير الجمعي: مجموع أي عدد ونظيره الجمعي يساوي صفر

قوانين الضرب والقسمة
a × b = b × a

خاصية التبديل للضرب: ترتيب الضرب لا يغير من النتيجة

(a × b) × c = a × (b × c)

خاصية التجميع للضرب: يمكن تجميع العوامل بأي ترتيب

a × 1 = a

العنصر المحايد للضرب: الضرب في واحد لا يغير القيمة

a × (1/a) = 1, where a ≠ 0

النظير الضربي: حاصل ضرب أي عدد غير صفر في مقلوبه يساوي واحد

خاصية التوزيع
a(b + c) = ab + ac

خاصية توزيع الضرب على الجمع: الضرب في مجموع يساوي مجموع نواتج الضرب

a(b - c) = ab - ac

خاصية توزيع الضرب على الطرح: الضرب في الفرق يساوي الفرق بين نواتج الضرب

الأسس والجذور
قوانين الأسس
a^m × a^n = a^(m+n)

ضرب قوى متماثلة الأساس: نجمع الأسس

a^m ÷ a^n = a^(m-n)

قسمة قوى متماثلة الأساس: نطرح الأسس

(a^m)^n = a^(m×n)

قوة القوة: نضرب الأسس

(a×b)^n = a^n × b^n

قوة الضرب: نوزع الأس على كل عامل

(a/b)^n = a^n / b^n

قوة القسمة: نوزع الأس على البسط والمقام

a^0 = 1 (a ≠ 0)

أي عدد غير صفري مرفوع للقوة صفر يساوي واحد

a^(-n) = 1/(a^n)

a^(1/n) = ∛a

الأس المقلوب: الجذر

a^(m/n) = (∛a)^m = ∛(a^m)

√(a × b) = √a × √b

جذر حاصل الضرب

جذر التكعيب

كثيرات الحدود
صيغ المربعات والمكعبات
(a + b)² = a² + 2ab + b²

مربع ثنائي الحد (المجموع)

(a - b)² = a² - 2ab + b²

مربع ثنائي الحد (الفرق)

(a + b)(a - b) = a² - b²

مجموع × فرق (فرق المربعين)

(a + b)³ = a³ + 3a²b + 3ab² + b³

مكعب المجموع

(a - b)³ = a³ - 3a²b + 3ab² - b³

مكعب الفرق

صيغ إضافية للأجزاء
a³ + b³ = (a + b)(a² - ab + b²)

مجموع المكعبات

a³ - b³ = (a - b)(a² + ab + b²)

فرق المكعبات

(a + b + c)² = a² + b² + c² + 2ab + 2ac + 2bc

مربع ثلاثي الحد

(a + b)(c + d) = ac + ad + bc + bd

ضرب ثنائيين

(a + b)(a² - ab + b²) = a³ + b³

مجموع المكعبات بصيغة أخرى

التحليل إلى عوامل
القوانين الأساسية للتحليل
a² - b² = (a + b)(a - b)

فرق المربعين

a² + 2ab + b² = (a + b)²

المربع الكامل (المجموع)

a² - 2ab + b² = (a - b)²

المربع الكامل (الفرق)

a³ - b³ = (a - b)(a² + ab + b²)

فرق المكعبات

a³ + b³ = (a + b)(a² - ab + b²)

مجموع المكعبات

تحليل كثيرات الحدود من الدرجة الثانية
ax² + bx + c = a(x - r₁)(x - r₂)

الصيغة العامة (r₁, r₂ هما جذور المعادلة)

x² + bx + c = (x + p)(x + q), where p + q = b and p × q = c

طريقة التجربة والخطأ (مجموع المعاملات = معامل x، حاصل الضرب = الثابت)

ax² + bx + c = a(x + b/2a)² + (c - b²/4a)

إكمال المربع

تحليل متعدد الحدود
P(x) = a(x - r₁)(x - r₂)...(x - rₙ)

الصيغة العامة (r₁, r₂, ... rₙ هي الجذور)

P(r) = 0 ⟺ (x - r) is a factor of P(x)

نظرية الباقي: إذا كان P(r) = 0، فإن (x - r) هو أحد عوامل P(x)

If P(x) = (x - r)Q(x) + R, then P(r) = R

نظرية الباقي العامة

المعاملات الحدية
If P(x) = aₙxⁿ + ... + a₁x + a₀, and r is a root, then r divides a₀

نظرية المعامل الثابت: جذور كثير الحدود P(x) هي من بين عوامل المعامل الثابت a₀

If P(x) = aₙxⁿ + ... + a₁x + a₀, and r = p/q in lowest form, then p divides a₀ and q divides aₙ

المعادلات
معادلات الدرجة الأولى
ax + b = 0

x = -b/a

الصيغة العامة والحل

a₁x + b₁ = a₂x + b₂

x = (b₂ - b₁)/(a₁ - a₂)

معادلة بمتغير واحد على الطرفين

معادلات الدرجة الثانية
ax² + bx + c = 0

x = (-b ± √(b² - 4ac)) / 2a

الصيغة القياسية وقانون حل المعادلة التربيعية (القانون العام)

Discriminant (Δ) = b² - 4ac

Δ > 0: two real roots

Δ = 0: one repeated real root

Δ < 0: two complex conjugate roots

المميز وطبيعة الجذور

For ax² + bx + c = 0 with roots r and s:

Sum of roots: r + s = -b/a

Product of roots: r × s = c/a

علاقة الجذور بالمعاملات

أنظمة المعادلات
For a system of two linear equations:

a₁x + b₁y = c₁

a₂x + b₂y = c₂

The solution is:

x = (c₁b₂ - c₂b₁)/(a₁b₂ - a₂b₁)

y = (a₁c₂ - a₂c₁)/(a₁b₂ - a₂b₁)

حل نظام معادلتين خطيتين بمتغيرين (طريقة كرامر)

Determinant (D) = a₁b₂ - a₂b₁

D ≠ 0: unique solution

D = 0: no solution or infinitely many solutions

المتباينات
قواعد المتباينات
If a < b, then a + c < b + c

إضافة نفس العدد للطرفين لا تغير اتجاه المتباينة

If a < b and c > 0, then a × c < b × c

ضرب الطرفين بعدد موجب لا يغير اتجاه المتباينة

If a < b and c < 0, then a × c > b × c

ضرب الطرفين بعدد سالب يعكس اتجاه المتباينة

If a < b and c < d, then a + c < b + d

جمع متباينتين متجهتين في نفس الاتجاه

حل المتباينات
ax + b < 0

x < -b/a (if a > 0)

x > -b/a (if a < 0)

متباينة من الدرجة الأولى

ax² + bx + c > 0

لحل هذه المتباينة، نحدد أولاً جذور المعادلة ax² + bx + c = 0، ثم نحدد فترات الموجب والسالب اعتماداً على إشارة a.

|ax + b| < c (where c > 0)

-c < ax + b < c

-c - b < ax < c - b

(-c - b)/a < x < (c - b)/a (if a > 0)

(c - b)/a < x < (-c - b)/a (if a < 0)

الدوال
الدوال الأساسية
Linear: f(x) = mx + b

Domain: ℝ, Range: ℝ

الدالة الخطية: m هو الميل، b هو نقطة التقاطع مع المحور الصادي

Quadratic: f(x) = ax² + bx + c

Domain: ℝ

Range: [vertex value, ∞) or (-∞, vertex value]

Vertex: (-b/2a, f(-b/2a))

الدالة التربيعية: القطع المكافئ

Cubic: f(x) = ax³ + bx² + cx + d

Domain: ℝ, Range: ℝ

الدالة التكعيبية

Exponential: f(x) = a•b^x, where b > 0, b ≠ 1

Domain: ℝ

Range: (0, ∞) if a > 0; (-∞, 0) if a < 0

الدالة الأسية

Logarithmic: f(x) = log_b(x), where b > 0, b ≠ 1

Domain: (0, ∞)

Range: ℝ

الدالة اللوغاريتمية

الدوال المثلثية
Sine: f(x) = sin(x)

Domain: ℝ, Range: [-1, 1]

Period: 2π

دالة الجيب

Cosine: f(x) = cos(x)

Domain: ℝ, Range: [-1, 1]

Period: 2π

دالة جيب التمام

Tangent: f(x) = tan(x)

Domain: x ≠ (n + 1/2)π, n ∈ ℤ

Range: ℝ

Period: π

دالة الظل

Cotangent: f(x) = cot(x)

Domain: x ≠ nπ, n ∈ ℤ

Range: ℝ

Period: π

دالة ظل التمام

General form: f(x) = A•sin(B(x-C)) + D or A•cos(B(x-C)) + D

A: amplitude

B: related to period (period = 2π/|B|)

C: phase shift

D: vertical shift

العمليات على الدوال
(f + g)(x) = f(x) + g(x)

مجموع دالتين

(f - g)(x) = f(x) - g(x)

فرق دالتين

(f × g)(x) = f(x) × g(x)

جداء دالتين

(f/g)(x) = f(x)/g(x), where g(x) ≠ 0

قسمة دالتين

(f ∘ g)(x) = f(g(x))

تركيب دالتين

الدالة العكسية
If f(x) = y, then f^(-1)(y) = x

A function has an inverse iff it is one-to-one

تعريف الدالة العكسية

f(f^(-1)(x)) = x and f^(-1)(f(x)) = x

خاصية التعويض المتبادل

Domain of f^(-1) = Range of f

Range of f^(-1) = Domain of f

مجال ومدى الدالة العكسية

If (a, b) is a point on f, then (b, a) is a point on f^(-1)
